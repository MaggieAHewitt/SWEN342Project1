The program is to be compiled as a java project with each class in the src folder. 
The number of developers is to be updated in the main.java if employees are added or removed
The % chance of a question being answered can be updated in the main.java.

Each time a meeting is held, the members of the meeting announce they are starting the meeting. When
a question is asked, if a team lead can answer it, that is announced, otherwise if the manager is free
they will answer the question. When anyone goes to lunch, there is an announcement, and the manager announces
when the group standup meeting begins. Each employee then announces their own departure.
